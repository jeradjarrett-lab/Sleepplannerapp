# Quick Tips: Maximizing Google Discover & Featured Snippets

## Immediate Actions After Deployment

### 1. Submit to Google Search Console
```
1. Add your site to Google Search Console
2. Submit your sitemap
3. Request indexing for all three main pages:
   - https://sleeplogic.app (Sleep Calculator)
   - https://sleeplogic.app/sleep-by-age (Sleep by Age)
   - https://sleeplogic.app/jet-lag (Jet Lag Calculator)
```

### 2. Create High-Quality Images
**Required Images:**
- `/og-image.png` (1200x630px) - Main Open Graph image
- `/logo.png` (512x512px) - Logo for publisher schema
- Optional: Section-specific og-images for each calculator

**Image Best Practices:**
- Use your brand colors (#4f86f7 blue, dark gradient)
- Include text overlay with value proposition
- High resolution, optimized for web
- Show the calculator interface in action

### 3. Validate Structured Data
```
Use these tools to validate your schema markup:
1. Google Rich Results Test: https://search.google.com/test/rich-results
2. Schema Markup Validator: https://validator.schema.org/
3. Google Search Console > Enhancements
```

## Monitoring & Analytics

### Key Metrics to Track
```
Google Search Console:
- Total impressions for Featured Snippets
- Position #1-3 rankings
- CTR for snippet queries
- Discover impressions and clicks

Google Analytics:
- Traffic from google.com/discover
- Bounce rate on landing pages
- Time on page
- Pages per session
```

### Search Console Filters
```
1. Performance > Search Appearance > "Rich Results"
2. Filter by "Position" < 4 (top 3 results)
3. Check "Discover" tab for Google Discover traffic
```
